ES6学习笔记（一）

發表於 2018-05-18 | 分類於 学习笔记

关于学习ES6语法中的块级作用域、let、const、模板字符串、标签模板的笔记。

块级作用域

以前只有函数作用域和全局作用域
块级作用域只在一段代码之间有效 {}（）括号内 如if function for switch等等

原本想用父级变量，但因为{}内声明了变量，变量提升，所以需要块级作用域

let

  • 不存在变量提升 必须声明后再使用
  • 可识别块级作用域 只在当前块级作用域内有效 （暂时性死区）
  • 不能重复声明（var 声明会覆盖前面 有可能会出错 所以建议先声明 后面用的时候再赋值）

const

  • 常量 不能改变
  • 具有let变量的性质 不提升 暂时性死区 不能重复声明
  • const obj = {} obj.a = 1; delete obj.a 对象可以改变
  • const A = 123,A拿到的是123占用的内存地址 原始值一旦改变需要重新分配内存 引用值改变地址不变

变量的结构赋值 模式匹配

数组

  • let [x = 10]=[undefined] 如果绝对等于undefined 就是默认值10
  • let [x = 10]=[null] 如果不是Undefined 就模式匹配 赋值为null
  • let [x=1,y=x]=[2] x=2 y=2 先模式匹配再默认值
  • let [x=y,y=1]=[] 后let y 但是在上面先x=y使用了y 所以会报错
  • let [x=y,y=1]=[2] 不会报错

对象

  • let{a,b} = {b:1,a:1} 无序
  • let{a:b} = {a:1} 输出a:undefined b:1
  • let{a,b}实质是let{a:a,b:b} 前面一个a是模式
  • let{toString:s} = false s==Boolean.prototype.toString

用于swap交换

  • 一般会临时变量交换
  • let a = 1,b = 2 [a,b] = [b,a]
  • 用于函数返回值的结构

模板字符串

增强的字符串 用包裹

  • 普通字符串换行\n 代码中\换行 实际输出会有很多空格
    模板字符串可以保留代码中的格式 语义化更好

  • ${}模板字符串中添加变量

标签模板

  • 函数调用的特殊形式 show a 相当于执行函数 show(a);

  • 过滤html字符串 视频没看懂 具体如何防注入？？

    • xss攻击 防注入 如在输入中输入了一些恶意脚本代码
    • 把字符串当作函数参数接收 通过正则匹配将脚本代码替换成别的让它无法执行

函数参数的默认值

  • ES6之前传入参数布尔值为False

  • es6中 可以直接写默认值

  • 与结构赋值结合使用 参数用结构赋值形式表示 如参数是对象 function fn({x,y=5}){} fn({x:1})

  • 参数作用域及注意事项

    • 传入两个相同的参数 相当于var了两遍 会取后面的一个
    • 传入两个相同的 再用一个默认值会报错 因为默认值就用的是Let let不能声明两遍
